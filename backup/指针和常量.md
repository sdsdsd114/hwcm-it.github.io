这张PPT讲的是C语言中指针与常量的两种核心组合：常量指针和指针常量，我帮你拆解一下👇
① 常量指针（const char *p）

• 定义：const char *p = "Hello";

• 含义：指针 p 本身是变量，但它指向的内容是常量，不能通过指针修改指向的内容。

• 代码分析：

◦ p[2] = 't'; → 错误 ❌
原因：p 指向的字符串 "Hello" 是常量，不能通过指针修改它的字符内容。

◦ p = "wang"; → 正确 ✅
原因：p 本身是变量，可以让它指向另一个字符串 "wang"。
② 指针常量（char *const p）

• 定义：char *const p = "Hello";

• 含义：指针 p 本身是常量，不能修改它的指向，但它指向的内容是变量，可以修改。

• 代码分析：

◦ p[2] = 't'; → 正确 ✅
原因：可以修改 p 指向的字符串里的字符（前提是字符串本身可写）。

◦ p = "wang"; → 错误 ❌
原因：p 是常量指针，不能让它再指向其他字符串。
快速记忆口诀

• const 在 * 左边：指向的内容是常量 → 常量指针（不能改内容，能改指向）

• const 在 * 右边：指针本身是常量 → 指针常量（能改内容，不能改指向）

• const 两边都有：const char *const p → 内容和指向都不能改
