# 树状数组（Binary Indexed Tree，Fenwick Tree）详解

## 1. 树状数组是什么？

树状数组是一种用于高效处理**前缀和**和**单点更新**的数据结构。它可以在 O(log n) 时间内完成：
- 查询任意前缀和
- 更新任意位置的值

## 2. 为什么要用树状数组？

### 对比其他方法：

| 操作 | 普通数组 | 前缀和数组 | 线段树 | **树状数组** |
|------|---------|-----------|--------|-------------|
| 单点更新 | O(1) | O(n) | O(log n) | **O(log n)** |
| 区间查询 | O(n) | O(1) | O(log n) | **O(log n)** |
| 实现复杂度 | 简单 | 简单 | 复杂 | **中等** |
| 空间 | O(n) | O(n) | O(4n) | **O(n)** |

树状数组在**实现简单**和**效率高**之间取得了很好的平衡。

## 3. 核心思想

树状数组的核心思想是：**每个位置存储的不是单个元素，而是某个区间的和**。

### 3.1 关键概念：lowbit

```cpp
lowbit(x) = x & (-x)  // 得到x二进制中最低位的1所代表的值
```

**例子**：
- x=6 (110): lowbit=2 (010)
- x=7 (111): lowbit=1 (001)
- x=8 (1000): lowbit=8 (1000)

### 3.2 存储结构

对于数组 a[1..n]，树状数组 BIT[i] 存储的是：
```
BIT[i] = sum of a[i - lowbit(i) + 1 ... i]
```

**直观理解**：
```
索引: 1    2    3    4    5    6    7    8
BIT: [a1] [a1+a2] [a3] [a1..a4] [a5] [a5+a6] [a7] [a1..a8]
      1     2      1      4      1      2      1      8    ← lowbit值
```

## 4. 基本操作详解

### 4.1 单点更新 update

```cpp
void update(int idx, int delta) {
    while (idx <= n) {
        BIT[idx] += delta;
        idx += lowbit(idx);  // 移动到父节点
    }
}
```

**图解更新过程**（更新位置3，加1）：

```
初始：BIT = [1, 2, 1, 4, 1, 2, 1, 8]
更新位置3：
第1步：idx=3，BIT[3]+=1 → [1,2,2,4,1,2,1,8]
第2步：idx=3+1=4，BIT[4]+=1 → [1,2,2,5,1,2,1,8]
第3步：idx=4+4=8，BIT[8]+=1 → [1,2,2,5,1,2,1,9]
第4步：idx=16>n，结束
```

### 4.2 前缀和查询 query

```cpp
int query(int idx) {
    int sum = 0;
    while (idx > 0) {
        sum += BIT[idx];
        idx -= lowbit(idx);  // 移动到前一个区间
    }
    return sum;
}
```

**图解查询过程**（查询前缀和到位置7）：

```
第1步：idx=7，sum+=BIT[7]=1，idx=7-1=6
第2步：idx=6，sum+=BIT[6]=2，sum=3，idx=6-2=4
第3步：idx=4，sum+=BIT[4]=5，sum=8，idx=4-4=0
结束，返回8
```

## 5. 完整示例

### 初始化
```cpp
n = 8
a = [3, 1, 4, 1, 5, 9, 2, 6]

构建BIT的过程：
初始BIT全0
for i=1 to 8:
    update(i, a[i])
```

构建过程：
```
i=1: update(1,3) → BIT[1]=3, BIT[2]=3, BIT[4]=3, BIT[8]=3
i=2: update(2,1) → BIT[2]=4, BIT[4]=4, BIT[8]=4
i=3: update(3,4) → BIT[3]=4, BIT[4]=8, BIT[8]=8
i=4: update(4,1) → BIT[4]=9, BIT[8]=9
i=5: update(5,5) → BIT[5]=5, BIT[6]=5, BIT[8]=14
i=6: update(6,9) → BIT[6]=14, BIT[8]=23
i=7: update(7,2) → BIT[7]=2, BIT[8]=25
i=8: update(8,6) → BIT[8]=31

最终BIT = [3, 4, 4, 9, 5, 14, 2, 31]
```

### 查询操作
```cpp
query(5) = a1+a2+a3+a4+a5 = 3+1+4+1+5 = 14
查询过程：
idx=5: sum+=BIT[5]=5, idx=5-1=4
idx=4: sum+=BIT[4]=9, sum=14, idx=4-4=0
返回14 ✅
```

## 6. 常见应用场景

### 6.1 求逆序对
```cpp
int countInversions(vector<int>& arr) {
    // 离散化
    vector<int> temp = arr;
    sort(temp.begin(), temp.end());
    for(int& x : arr) {
        x = lower_bound(temp.begin(), temp.end(), x) - temp.begin() + 1;
    }
    
    int n = arr.size();
    vector<int> BIT(n+1, 0);
    int ans = 0;
    
    for(int i = n-1; i >= 0; i--) {
        ans += query(arr[i] - 1);  // 统计比当前数小的个数
        update(arr[i], 1);
    }
    return ans;
}
```

### 6.2 动态第K小（正是本题的应用）
```cpp
int findKth(int k) {
    int left = 1, right = n, ans = -1;
    while(left <= right) {
        int mid = (left + right) / 2;
        if(query(mid) >= k) {
            ans = mid;
            right = mid - 1;
        } else {
            left = mid + 1;
        }
    }
    return ans;
}
```

## 7. 树状数组 vs 线段树

| 特性 | 树状数组 | 线段树 |
|------|---------|--------|
| 代码长度 | 短（~20行） | 长（~50行） |
| 时间复杂度 | O(log n) | O(log n) |
| 空间复杂度 | O(n) | O(4n) |
| 支持的操作 | 前缀和、单点更新 | 区间更新、区间查询等 |
| 适用范围 | 较窄 | 广泛 |

**选择建议**：
- 只需要前缀和/单点更新 → 用树状数组
- 需要区间更新/复杂查询 → 用线段树

## 8. 进阶：二维树状数组

```cpp
class BIT2D {
    vector<vector<int>> bit;
    int n, m;
    
public:
    BIT2D(int n, int m) : n(n), m(m) {
        bit.resize(n+1, vector<int>(m+1, 0));
    }
    
    void update(int x, int y, int val) {
        for(int i = x; i <= n; i += i&-i)
            for(int j = y; j <= m; j += j&-j)
                bit[i][j] += val;
    }
    
    int query(int x, int y) {
        int sum = 0;
        for(int i = x; i > 0; i -= i&-i)
            for(int j = y; j > 0; j -= j&-j)
                sum += bit[i][j];
        return sum;
    }
};
```

## 9. 常见坑点和技巧

### 9.1 下标从1开始
树状数组通常从1开始，避免处理0的特殊情况。

### 9.2 离散化
当数值范围很大时（如10^9），需要先离散化：
```cpp
vector<int> nums = {100, 200, 50, 300};
// 离散化后：{2, 3, 1, 4}
```

### 9.3 区间查询转前缀和
```cpp
// 查询[l, r]区间和
int rangeSum(int l, int r) {
    return query(r) - query(l-1);
}
```

## 10. 总结

树状数组的精髓：
1. **巧妙利用二进制**：通过lowbit实现快速跳转
2. **空间换时间**：用O(n)空间换O(log n)时间
3. **前缀和思想**：任何区间和都可以转化为两个前缀和之差

**记忆口诀**：
- 更新：`i += lowbit(i)` 往上爬
- 查询：`i -= lowbit(i)` 往下跳
- lowbit：`x & -x` 找最低位的1

这就是为什么树状数组能在保持代码简洁的同时，达到和线段树相近的效率！
